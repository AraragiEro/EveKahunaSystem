# 常见问题

## 为什么我的材料算不准？

材料的加成计算涉及以下：
- 建筑本身
    - 莱塔卢 索提约 阿兹贝尔有1%的材料加成，在选择建筑分配后会生效，这个一般不会出错。
- 建筑插
    - 建筑插加成会在配置了对应的插件配置和分配了建筑后生效，一般不会出错。
- 蓝图
    - 在配置了缺省蓝图系数后生效，一般不会出错。
- 最大轮数， 问题一般出在这。
    - ![最大轮数配置图](/assets/doc-QA-1.png)
    - ![计划配置开关](/assets/doc-QA-2.png)
    - 在开启了工作流切分且配置了最大轮数后，工作会进行对应切分。切分后会导致材料系数无法吃满，结果就是总材料需求增加。
    - ![工作流图例](/assets/doc-QA-3.png)
    - 工作流是具备执行能力的任务表。可以在工作流中确认你的工作切分状态。
