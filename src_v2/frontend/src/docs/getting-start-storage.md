# 库存管理
库存管理需要alpha订阅,允许通过esi拉取公司或个人库存数据，用于进行更高精度和智能工业计划中的库存计算。

## 资产浏览
在资产浏览界面可以配置某个访问许可的目标容器的监控视图。
视图目前有3种
- 默认
    可以显示物品种类和数量
- 销售
    会拉取物品的价格，针对销售行为做优化。可以生成共享链接让任何人访问
- 监控
    用来监控某类物品是否低于库存警戒线

## 资产拉取
![资产拉取图例](/assets/doc-storage-1.png)
此处可以添加资产拉取任务，以及手动更新资产信息。

需要总监权限才可添加公司资产拉取。

最短每5分钟可更新一次。

## 访问许可
![资产拉取图例](/assets/doc-storage-2.png)
在访问许可通过搜索特定的物品锁定容器id后即可给容器添加标签。

在进行此步骤前建议携带若干牌子等不会被生产消耗的物品放特定的数量到你的每一个箱子里。
